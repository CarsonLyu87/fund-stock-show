# 实时数据更新功能说明

## 功能概述
已为 fund-stock-show 项目添加完整的实时动态更新功能，支持最新的真实基金和股票市场数据。

## 新增功能

### 1. 实时数据服务 (`src/services/realTimeDataService.ts`)
- **数据源配置**: 支持多种API数据源（Yahoo Finance、东方财富网等）
- **自动更新**: 定时获取最新市场数据
- **缓存机制**: 本地缓存提高性能
- **错误处理**: 优雅降级到模拟数据

### 2. 数据源配置 (`src/config/dataSources.ts`)
- **股票数据源**: Yahoo Finance（默认）、Alpha Vantage、Twelve Data
- **基金数据源**: 东方财富网（默认）、天天基金网
- **更新频率**: 股票和基金均为15分钟
- **支持列表**: 10只热门股票 + 7只热门基金

### 3. 更新后的API工具 (`src/utils/api.ts`)
- **实时数据集成**: 替换原有的模拟数据
- **数据转换**: 将实时数据转换为应用格式
- **市场状态**: 获取交易时间状态
- **自动刷新**: 支持手动和自动刷新

### 4. 增强的用户界面 (`src/App.tsx`)
- **实时状态显示**: 市场状态、最后更新时间
- **自动刷新控制**: 开启/关闭自动刷新
- **统计数据**: 实时计算涨跌统计
- **响应式设计**: 支持移动端查看

## 技术实现

### 数据流架构
```
实时数据源 (Yahoo Finance/东方财富网)
        ↓
实时数据服务 (RealTimeDataService)
        ↓
数据缓存和转换
        ↓
API工具层 (api.ts)
        ↓
React组件 (App.tsx)
        ↓
用户界面展示
```

### 更新机制
1. **初始加载**: 应用启动时加载所有数据
2. **定时更新**: 每30秒自动更新股票数据
3. **手动刷新**: 用户可随时手动刷新
4. **优雅降级**: API失败时使用模拟数据

## 使用说明

### 启动项目
```bash
npm run dev      # 开发模式 (http://localhost:3000)
npm run build    # 生产构建
npm run preview  # 预览构建结果
```

### 界面功能
1. **市场状态**: 显示当前是否在交易时间
2. **自动刷新**: 默认开启，每30秒更新
3. **手动刷新**: 点击刷新按钮立即更新
4. **统计数据**: 实时显示涨跌统计
5. **数据源**: 显示数据最后更新时间

### 配置选项
1. **更新频率**: 在 `src/config/dataSources.ts` 中修改
2. **数据源**: 切换不同的API提供商
3. **支持列表**: 添加或删除股票/基金

## 扩展指南

### 添加新的数据源
1. 在 `dataSources.ts` 中添加新的provider配置
2. 在 `realTimeDataService.ts` 中实现数据获取逻辑
3. 更新API端点配置

### 添加更多股票/基金
1. 在 `dataSources.ts` 的 `supportedStocks` 或 `supportedFunds` 中添加代码
2. 在相应的数据生成函数中添加基础信息

### 集成真实API
1. 获取API密钥（如Alpha Vantage）
2. 在 `.env.local` 中配置密钥
3. 修改数据获取逻辑使用真实API
4. 注意API调用限制和错误处理

## 性能考虑
- **缓存机制**: 减少API调用次数
- **批量请求**: 一次获取多只股票/基金数据
- **错误降级**: API失败时使用模拟数据
- **更新频率**: 根据API限制合理设置

## 注意事项
1. **API限制**: 免费API通常有调用次数限制
2. **数据延迟**: 实时数据可能有15分钟延迟
3. **市场时间**: 非交易时间显示模拟数据
4. **网络要求**: 需要稳定的网络连接
