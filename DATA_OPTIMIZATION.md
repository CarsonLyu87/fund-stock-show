# 数据优化说明

## 🎯 优化目标
将 `fund-stock-show` 项目的数据更新频率从30秒改为每小时更新一次，并集成真实数据源。

## 🔧 主要优化内容

### 1. 更新频率调整
- **之前**: 每30秒自动刷新
- **现在**: 每小时（3600000毫秒）自动刷新
- **位置**: `src/App.tsx` 中的 `useEffect` 自动刷新间隔

### 2. 真实数据服务集成
创建了新的数据服务文件：`src/services/realDataService.ts`

#### 支持的真实数据源：
- **基金数据**:
  - 天天基金实时估值API
  - 东方财富基金详情API
  - 天天基金净值API

- **股票数据**:
  - 雅虎财经API（国际股票）
  - 新浪财经API（A股数据）
  - 腾讯财经API

#### 数据缓存机制：
- 本地内存缓存
- 缓存存活时间：30分钟
- 自动缓存失效和更新

### 3. 降级策略
当真实API不可用时，自动降级到模拟数据：
- 生成基于市场规律的模拟数据
- 保持数据结构的完整性
- 提供合理的市场波动

### 4. 配置优化
更新了 `src/config/dataSources.ts`：
- 调整更新频率配置
- 添加缓存配置选项
- 优化API端点配置

## 🛠️ 部署说明

### 环境变量
创建 `.env.local` 文件配置：
- API密钥（如果需要）
- 代理设置（绕过CORS）
- 更新频率配置
- 缓存配置

### 构建和部署
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建
npm run build

# 部署到GitHub Pages
npm run deploy
```

### 注意事项
1. **CORS问题**: 部分API可能需要代理或后端转发
2. **API限制**: 免费API有调用频率限制
3. **数据延迟**: 真实数据可能有1-2秒延迟
4. **市场时间**: 注意交易时间和闭市时间
